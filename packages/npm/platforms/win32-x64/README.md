# @opendsr/seedfaker-win32-x64

Platform binary for [@opendsr/seedfaker](https://www.npmjs.com/package/@opendsr/seedfaker).

**Do not install this package directly.** It is an optional dependency installed automatically based on your OS and CPU architecture.

## Install

```
npm install @opendsr/seedfaker        # Node.js library
npm install @opendsr/seedfaker-cli    # CLI tool
```
